# @quorum/shared

Shared types, utilities, and constants for the Quorum application.

## Usage

```typescript
import { generateId, formatDate, type BaseEntity } from "@quorum/shared";
import { delay } from "@quorum/shared/utils";
import { type PaginatedResponse } from "@quorum/shared/types";
```

## Contents

- `types/` - Common TypeScript types and interfaces
- `utils/` - Utility functions used across the application
